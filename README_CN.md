# Rust Weather 项目

## 概述
Rust Weather 是一个使用 Rust 编程语言编写的天气查询项目。

## 安装
要在您的计算机上安装并运行该项目，请按照以下步骤操作：
1. 克隆仓库：
   ```bash
   git clone https://github.com/Mo2hu/rust-weather.git
   ```
2. 进入项目目录：
   ```bash
   cd rust-weather
   ```
3. 构建项目：
   ```bash
   cargo build
   ```
4. 运行项目：
   ```bash
   cargo run
   ```

## 功能
- 实时天气查询
- 多城市支持
- 可自定义的通知设置

## 使用说明
运行程序后，您可以输入城市名称以获取当前天气信息。
