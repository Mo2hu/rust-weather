# Rust Weather

## Project Documentation
Rust Weather is a simple yet effective weather application built using Rust. It fetches real-time weather data and presents it in a user-friendly format.

## Setup Instructions
1. **Clone the repository:**  
   ```bash
   git clone https://github.com/Mo2hu/rust-weather.git
   cd rust-weather
   ```  

2. **Install dependencies:**  
   Ensure you have Rust installed. If not, you can install it from [rust-lang.org](https://www.rust-lang.org/).
   ```bash
   cargo build
   ```  

3. **Environment Variables:**  
   Set the necessary environment variables for API keys (if any).

## Usage Guide
To run the application, use the following command:  
```bash
cargo run
```

You can customize the weather location by passing it as a command-line argument:  
```bash
cargo run <location>
```

Replace `<location>` with your desired city or geographical coordinate.
